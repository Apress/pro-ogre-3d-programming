OGRE (Object-Oriented Graphics Rendering Engine)

Demo Distribution README file

http://www.ogre3d.org

Summary

OGRE (Object-Oriented Graphics Rendering Engine) is a scene-oriented, flexible 3D engine written in C++ designed to make it easier and more intuitive for developers to produce games and demos utilising 3D hardware. The class library abstracts all the details of using the underlying system libraries like Direct3D and OpenGL and provides an interface based on world objects and other intuitive classes.

To run these demos you need to have DirectX 9 installed, although you can also choose to run the demos in OpenGL if you wish. 

So now what?

These demos are only a small sample of what you can achieve with OGRE. For more eye candy, visit the Official Web Site, and click on 'Featured Projects. If you would like to start making applications using OGRE, you can download the latest source code and documentation from the site
